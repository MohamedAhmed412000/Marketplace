Task Specifications

  • Database consists of 5 tables

    • User Table
       Id username email date_joined last_login is_admin is_staff is_superuser balance

    • Product Table
       Id title description price quantity recommend category  owner image

    • Cart Table
       Id  user  product quantity

    • Market Table
       Id  user  product

    • Ownership Table
       Id  seller  buyer  product quantity date_buy

  • Authentication done with inherited model from Base User model.

  • Integrated Payment Gateway (PayPal) is Added.




Features:

  • User can create account.

  • User can view all products on website.

  • User can add, edit, or delete product to/from his market to buy it.

  • User can search by product title or product category.

  • User can add, remove product to his cart.

  • User can recommend other uers products or remove the recommendation.

  • User can use PayPal for transaction.

  • User can view his balance, transaction history.
